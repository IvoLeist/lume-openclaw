# Docs

This folder contains the public documentation that complements the root [README](../README.md).

## Start Here

- [runbook-lume-openclaw-setup.md](runbook-lume-openclaw-setup.md): full end-to-end setup, first boot, OpenClaw install, channels, headless operation, updates, and troubleshooting
- [port-forwarding.md](port-forwarding.md): exposing the VM gateway for Telegram, WhatsApp, and other webhook-based channels
- [peekaboo-bridge.md](peekaboo-bridge.md): enabling UI automation inside the VM with Peekaboo Bridge

## Suggested Reading Order

1. Read the root [README](../README.md) for the quickstart and the main host-side commands.
2. Use [runbook-lume-openclaw-setup.md](runbook-lume-openclaw-setup.md) for the full first-time setup.
3. Add [port-forwarding.md](port-forwarding.md) if you need public webhooks.
4. Add [peekaboo-bridge.md](peekaboo-bridge.md) if you want macOS UI vision and automation inside the VM.

## Scope

The docs kept here are the ones that should help a new repo user operate the project. Internal planning notes, one-off audit dumps, and temporary checklists have been removed so the folder stays publishable.
